A [[gameplayfeature:drop-targets]] variant in which the target face contains integrated lighting, allowing it to display colors or patterns that communicate game state to the player. The illumination can indicate which targets are active, highlight completion progress, or signal timed scoring opportunities — adding a visual feedback layer that conventional printed-insert drop targets lack. [[manufacturer:spooky]]'s *[[title:rick-and-morty]]* (2020) featured two illuminating drop target banks, using the lit targets to convey mode status and scoring multipliers directly on the target face rather than relying solely on playfield inserts or the backbox display.
