---
name: Ball Return Gates
aliases:
  - Ball return gate
---

One-way gates on the playfield that allow the ball to pass through in one direction while blocking it from returning. Typically mounted at lane exits or outlane boundaries, ball return gates route the ball along a designed path without the need for electrical control. They appear frequently on [[technologygeneration:electromechanical]] machines of the 1960s and 1970s, where manufacturers like [[manufacturer:bally]] and [[manufacturer:williams]] used them to create outlane kickback paths, upper-playfield return routes, and controlled drainage channels. A purely mechanical device — a metal flap on a pivot — the ball return gate remains one of pinball's simplest and most enduring playfield elements.
