name: Virtual Kickbacks

A software-based ball save that rescues the ball without any physical mechanism in the outlane. When the feature is active and the ball drains through an outlane, the game's software registers the save and delivers a new ball to the plunger lane — mimicking the effect of a traditional [[gameplayfeature:kickback]] through code alone. Because no solenoid or magnet is involved, the rescue feels seamless and instantaneous. [[manufacturer:stern-pinball]]'s *[[title:venom]]* uses virtual kickbacks at both outlanes, where spelling SUIT at a target bank lights the save. The feature reflects modern pinball's increasing reliance on software to replicate what once required dedicated hardware.
