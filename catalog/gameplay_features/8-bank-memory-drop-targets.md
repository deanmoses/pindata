A bank of eight [[gameplay-feature:memory-drop-targets]] — [[gameplay-feature:drop-targets]] that remember their dropped state between turns, allowing incremental completion across multiple balls. IPDB catalogs a single machine with this configuration: *[[title:star-god]]* (1980) by [[manufacturer:zaccaria]]. The eight-target memory bank was the longest memory drop target sequence cataloged, and the carry-over mechanic was essential — clearing eight targets in a single ball would have been exceptionally difficult without it.
