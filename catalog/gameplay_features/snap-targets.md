A moving [[gameplayfeature:targets]] that alternates between two fixed positions, pivoting from one stationary state to the other each time the ball strikes it. Unlike [[gameplayfeature:swinging-targets]], which swing back and forth in continuous motor-driven motion, a snap target holds its position until hit and then locks into the opposite orientation. The player must plan shots based on which face the target currently presents, since the scoring value or game effect may differ between the two positions. Snap targets add a tactical layer to target shooting — every hit changes the state of the target for the next shot.
